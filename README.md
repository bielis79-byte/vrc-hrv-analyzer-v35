# VRC / HRV RRi Analyzer Pro v3.5

Aplicación Streamlit para análisis de variabilidad del ritmo cardíaco a partir de RRi.

## Incluye

- Segmentación flexible: Basal + E1-E6 + R1-R3.
- Ventanas de duración libre, no limitadas a 5 minutos.
- Cálculo por ventanas válidas con al menos 30 RRi.
- Métricas temporales, frecuenciales, no lineales, RQA, MSE y HVG.
- Poincaré por fase.
- Exportación a Excel, CSV, PNG y ZIP.

## Ejecutar localmente

```bash
streamlit run app.py
```

## Archivos necesarios para Streamlit Cloud

```text
app.py
requirements.txt
README.md
```
